# 盤面分層與空格

## 三層結構

| 層 | 代表物件 | 視覺 | 與中層元素 |
|----|----------|------|------------|
| **bottom** | 水窪 Puddle | 畫在格子底色上，約 72% 透明 | **仍可見**上方元素（除非中層圖太滿） |
| **middle** | 紙箱、桶、罐頭、櫃、泳池、元素、道具… | 主玩法層 | 本身即元素或佔格障礙 |
| **upper** | 繩索 Rope、泥巴 Mud | 疊在上層 | **繩索**：元素可見、可配對但鎖住不能動<br>**泥巴**：**完全遮住**中層元素 |

實作（Godot）：

- 水窪：`board_bg` 先畫半透明底圖；該格**不**進 `blocked_cells`，開局會填糖。
- 泥巴：`candy.visible = false`；`board_bg` 最後畫不透明 Mud。
- 繩索：`obstacle.type = wire` → 鎖 swap，match 可含該色但糖不消（對齊 Python）。

## 鮭魚罐 SalmonCan

| 屬性 | 值 |
|------|-----|
| HP | **2** |
| 開局 | `salmon_state = sealed`（未開蓋，略加深遮罩） |
| 消除 | **僅道具**（炸彈 / 火箭 / 紙飛機落點等 `SPECIAL`/`PLANE`） |
| 三消鄰邊 | **無效** |
| 流程 | 第 1 下道具 → 開蓋（`open`）<br>第 2 下道具 → 清除 |

## 盤內「空格」vs `void`

| 標記 | 意義 | 元素能否進入 |
|------|------|--------------|
| `void` | 盤面**外形外的洞**（非 playable） | 否，在 `blocked_cells` |
| `null` 中層 + 非 blocked | 盤內**可玩空格**（漏斗、障礙打掉後的洞） | **可以**落下、可滑動移入（需能形成 match） |

關卡 JSON：外形缺口用 `"void"`；盤內空腔用 `null`，不要用 `void`。

滑動：對鄰格空格可「移入」（非雙向 swap），無 match 則彈回。

## 可移動障礙物（木桶 / 交通錐）

與元素、道具相同規則：

- **滑動**：鄰格糖/道具 ↔ 桶或錐（須能形成 match，否則彈回）；也可**直接拖動**桶/錐。
- **點選**：先點桶/錐再點鄰格糖，或先點糖再點桶/錐。
- **重力**：消除後會落下，可斜落入盤內空格。
- 交通錐 `lv2` = 2 HP，仍可移動。
